# Odin Library

A dynamic Library application built with HTML, CSS, and JavaScript as part of the JavaScript path from **[The Odin Project](https://www.theodinproject.com)**.

This project focuses on practicing:

- Object constructors
- Arrays and object management
- DOM manipulation
- Event handling
- Form handling
- Data attributes
- Prototype methods
- Dynamic UI rendering
